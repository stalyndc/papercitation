# Paper Citation

Find academic sources for any claim. Instant citations in APA, MLA, and Chicago formats.

## Stack

- Frontend: Static HTML/CSS/JS on Cloudflare Pages
- Backend: Cloudflare Worker
- AI: Gemini API
- Data: OpenAlex API

## Development

```bash
cd worker
npm run dev
```

## Deployment

Worker deploys via `wrangler deploy`. Frontend deploys via Cloudflare Pages connected to GitHub.
